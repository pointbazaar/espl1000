Primitive Data Types

The primitive Data Types available in ESPL1000 are used similar to those available in C, and are a subset of those.

Integer Types

Type Width Signed-ness
int8 1 byte signed
int unspecified signed
uint8 1 byte unsigned
uint unspecified unsigned

char

char c = 'k';

bool

bool x = true;

Hex and Binary Literals

uint8 x;
x = 0x0f
x = 0b01000101
