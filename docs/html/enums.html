An enum in smalldragon is a type with zero or more named values of that type.

Rules

  • An enum must be named in uppercase snake case.
  • Enum values must be named in upprecase snake case.

Use Cases

To describe a type which is neither composite (a struct) nor a prmitive discrete type or boolean type.
Any type where there are a fixed, small number of values which can be named.

Example

enum DIR {
    UP; DOWN; LEFT; RIGHT;
}

enum PERMISSION {
    R; W; X;    RW; RX; WX;     RWX;
}

fn main() ~> int {
    DIR dir = DIR:UP;

    PERMISSION perm = PERMISSION:R;

    return dir == DIR:LEFT && perm == PERMISSION:RWX;
}
