Comparison Operator Chaining

Comparison Operator Chaining allows to express comparisons of more than 2 Variables / Constants in a succinct way.

Granted, chained comparisons are not that common, but not uncommon either. They can improve the readability of Code.

Supported Comparison Operators 

<, >, <=, >=, !=, ==

Example 

fn main () ~> int {

	count     = 4;
	count_max = 10;

	if 0 < count < count_max {
		return 0;
	}

	return count > 0;
}
